Recortar Imágenes a Círculo - Tamaño Grande

Instrucciones:

  1. Selecciona múltiples imágenes haciendo clic en "Elegir Imágenes"
  2. Elige el tamaño de salida (hasta 4000x4000 píxeles)
  3. Cada imagen se mostrará con un recorte circular previo
  4. Puedes arrastrar cada imagen para ajustar el recorte
  5. Haz clic en "Descargar esta imagen" para guardar individualmente
  6. Haz clic en "Descargar todas" para obtener un ZIP con todas las imágenes
