Recortar Múltiples Imágenes a Círculo 1200x1200

Instrucciones:

  1. Selecciona múltiples imágenes haciendo clic en "Elegir Imágenes"
  2. Cada imagen se mostrará con un recorte circular previo
  3. Puedes arrastrar cada imagen para ajustar el recorte
  4. Haz clic en "Descargar esta imagen" para guardar individualmente
  5. Haz clic en "Descargar todas" para obtener un ZIP con todas las imágenes (1200x1200 PNG)
